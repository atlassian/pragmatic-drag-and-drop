This page demonstrates some of the experiences that Pragmatic drag and drop enables. These examples
use a combination of our [core package](/components/pragmatic-drag-and-drop/core-package) and
[optional packages](/components/pragmatic-drag-and-drop/optional-packages).

## File

Use the [external adapter](/components/pragmatic-drag-and-drop/adapters/external) to capture files
that users drag and drop onto the page.

When requesting files, you should also provide an `<input type="file" />` for usability and
accessibility reasons.
